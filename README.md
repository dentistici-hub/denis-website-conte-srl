## Build

```bash
npm install
npm run dev      # local preview at http://localhost:<port>/denis-website-conte-srl/
npm run build    # static output in dist/
npm run preview  # serve built output
```

Automated via `.github/workflows/deploy.yml` on push to `main`. GitHub Pages serves the artifact built by the workflow.
